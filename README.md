# ❤️ AI-Powered Heart Disease Prediction System

An end-to-end Machine Learning web application that predicts the risk of heart disease using clinical patient data.

Built with **Python**, **Scikit-Learn**, and **Streamlit**.

---

## 📌 Project Overview

This application predicts whether a patient is at risk of heart disease based on clinical parameters.

The project includes:

- Professional Streamlit Web Application
- Machine Learning Pipeline
- Smart Risk Analysis
- Heart Health Score
- Prediction Confidence
- Personalized Recommendations
- Responsive User Interface

---

## 📊 Dataset Information

- Source: Kaggle Heart Disease Dataset
- Records: **918**
- Features: **11**
- Target Variable: **HeartDisease**

---

## 🤖 Machine Learning Model

| Model | Extra Trees Classifier |
|--------|------------------------|
| Accuracy | **91.30%** |

Hyperparameter tuning was performed using GridSearchCV.


Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
